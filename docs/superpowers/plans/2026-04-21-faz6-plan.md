### Task 3 — Streaming Chat (EN RİSKLİ)
**Mevcut:** Full request/response — tool use çalışıyor, stabil

**Sorun:** Tool use + streaming birlikte karmaşık:
- Text turn → stream edilebilir ✓
- Tool call turn → streaming desteklemez, full collect etmek gerekir
- Tool result + sonraki yanıt → yeniden stream

**Öneri:** Hibrit yaklaşım
1. API'de `stream: true` aç, `text` event'lerini SSE ile ilet
2. Tool call gelince stream'i durdur, tool'u çalıştır, yeni request yap
3. Client: EventSource ile satır satır göster

**Alternatif (daha basit):** Sadece son mesajı stream et (tool loop bittikten sonra)  
→ Kullanıcı bot "yazıyor..." görür, sonuç gelince satır satır akar

**Risk:** Yüksek — mevcut chat akışını bozabilir. **Opus 4.7 ile yap.**
